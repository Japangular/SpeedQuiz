architecture Tech Stack & Architecture

The application runs as a multi-container Docker Compose setup with five services: an Angular SPA served via nginx, a Spring Boot backend, a PostgreSQL database, a read-only SQLite mount for Anki imports, and a Python FastAPI microservice for Japanese text segmentation via MeCab.

Frontend

Angular 19 with Angular Material. State is managed via RxJS BehaviorSubjects in injectable services. The API client is auto-generated from an OpenAPI 3.0 contract to keep frontend and backend in sync.

Backend

Spring Boot 3.4 on Java 21. Uses Spring Data JPA with a multi-datasource configuration — PostgreSQL for application data and SQLite for read-only Anki deck imports. Database migrations are handled via Flyway. Session management is lightweight and passwordless, using UUID tokens.

NLP Service

A Python FastAPI service wrapping MeCab for morphological analysis of Japanese text. Called server-to-server from the Spring Boot backend. Containerized separately to keep the Java build lean.

Infrastructure

Docker Compose orchestrates all services. nginx acts as reverse proxy, serving the SPA and routing /api/* requests to the backend. Static assets like KanjiVG stroke order SVGs are served directly from nginx.

featured_play_list Notable Engineering Decisions

OpenAPI code generation — The API contract lives in a single api.yaml file. The Angular API client is generated from it, eliminating manual DTO maintenance and catching contract mismatches at build time.

Multi-datasource configuration — PostgreSQL handles application state while Anki's SQLite database is mounted read-only. Spring's qualifier-based injection keeps the two datasources cleanly separated.

Fuzzy answer validation — Quiz answers are validated using Levenshtein distance for typo tolerance, combined with a custom romaji-to-hiragana converter so users can type in either script.

Polyglot architecture — MeCab (the Japanese tokenizer) only has mature bindings for Python and C. Rather than fighting with JNI, I run it as a separate microservice — a pragmatic tradeoff between language ecosystem strengths.

 auto_stories About JapanGular

JapanGular is a Japanese learning tool built by a fan who wanted a better way to study vocabulary and kanji encountered in streams. Instead of relying on generic apps, this one is tailored to the content we actually watch.

Everything here is a personal project — not affiliated with any vtuber or agency. Emotes and assets are used with permission for non-commercial purposes.

help_outline How to Use

Quiz

Pick a deck and test yourself. Answers are checked with some typo tolerance, and you can type in either romaji or hiragana. Cards you get wrong will come back more often.

Dictionary

Paste any Japanese text and it gets broken down into individual words with dictionary lookups. Useful when you catch a word in a stream and want to look it up quickly.

Kanji Wall

A visual grid of kanji. Click any character to see its readings, meanings, and animated stroke order.

Transcripts

Upload stream transcripts and turn them into study material. The app parses the Japanese text and makes it searchable.

Anki Table

If you use Anki, you can browse your existing decks here. Rows you already know can be marked as ignored.

Deck Creator

Build your own flashcard decks with custom properties — questions, answers, hints, images, audio, whatever you need.
